How PackBot Works

1. Describe or Code

In simple mode, write what you want your bot to do. In code mode, edit files directly.

2. We Generate & Push

Your bot becomes a folder in a separate GitHub repo. Every file is committed instantly.

3. Run & Share

Preview your bot live, share the URL, or fork it to make a new version.
